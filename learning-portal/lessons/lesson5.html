Lesson 5

Hardware Wallets

Hardware wallets isolate private keys from internet-connected devices.

Trezor

Open-source hardware with intuitive UI and strong backup workflows.

Ledger

Secure element architecture with a polished companion app.

Coldcard

Air-gapped Bitcoin-only device emphasizing security-first operations.

Passphrases

BIP39 passphrases add a secret offset to the seed, creating additional protection.

PIN security

Device PINs limit physical access; duress PINs can wipe or decoy.

Summary

  • Use hardware wallets to separate signing keys from your daily computer.
  • Passphrases increase protection against seed compromise.
  • Test recovery with a small restore drill.
