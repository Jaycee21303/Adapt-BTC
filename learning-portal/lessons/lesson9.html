Lesson 9

Keys, Sats, Addresses

Understanding keys and address formats is essential for safe transactions.

Private/public keys

The private key signs; the public key verifies and derives addresses.

Address types

Legacy (P2PKH), SegWit (P2SH/P2WPKH), and Taproot (P2TR) differ in efficiency and features.

Bech32

SegWit bech32 addresses start with bc1 and reduce errors with checksum encoding.

Taproot

Taproot addresses enable key and script path spending with better privacy.

Derivation paths

BIP32/44/84 define how wallets derive many addresses from one seed.

Summary

  • Keep private keys secret; share only addresses.
  • Use bech32 or Taproot for efficiency.
  • HD wallets derive fresh addresses to protect privacy.
