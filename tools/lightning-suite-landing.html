Why We Built This

Bitcoin adoption can be overwhelming, especially for beginners and small businesses. The Lightning Network adds even more complexity — channels, liquidity, invoices, routing, fees, node selection, and payment failures.

The AdaptBTC Lightning Adoption Suite solves this with an integrated, zero-cost, browser-based tool that educates, analyzes, and guides users step-by-step.

What’s Inside

Lightning Invoice Decoder

Paste an invoice to reveal amount, memo, expiration, pubkey, and metadata.

Payment Health Checker

Checks node uptime, liquidity, and routing success probability for any invoice or pubkey.

Routing Path Visualizer

Interactive hop-by-hop diagram that shows how Lightning payments travel through the network.

Mempool Heat Map

Real-time network congestion data so users understand when fees are low or high.

Lightning Fee Calculator

Clear breakdown of expected fees based on amount and hop count.

Channel Profitability Modeler

Helps merchants calculate routing revenue, ROI, and optimal channel size.

Node Health Leaderboard

Ranks routing nodes by uptime, capacity, and trust score.

Lightning Geo Map

Shows global Lightning nodes visually for educational and analytical value.

Merchant Setup Advisor

Recommends channel setup, liquidity ratios, redundancy, and node selection.
